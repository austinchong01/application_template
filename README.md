Interview Assistant

Core Functionalities
1. Input Collection
    - Company website URL
    - Job posting URL
    - LinkedIn URL (optional, may fail)
2. Content Fetching
    - Scrape inputs
3. AI Analysis
    - Company
        - Industry
        - Age
        - Mission
        - Values
        - Size
        - Stage
        - Other culture indications
    - Job
        - Title
        - Team
        - Required Skills
        - Preferred Skills
        - Tech Stack (if technical role)
        - Experience Level
4. AI Web Search
    - Interview experiences (Reddit, Glassdoor, Blind, etc.)
    - Interview Questions
    - Company recent events
    - Current need of company
5. Overall Analysis
    - Potential Interview Questions
    - Questions to ask interviewer
    - What the company needs
    - How to be of impact
6. Output
    - Editable Document creation

Nice-to-Have Functionalities
1. User Accounts
2. PDF Export
3. Recruiter Look-up
4. Database
